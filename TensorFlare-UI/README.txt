Introduction:
-------------
TensorFlare UI is an interactive application designed to allow users to engage with the TensorFlare chess engine. The UI offers an intuitive interface for chess enthusiasts to play against the Manual-Tensor engine, a variant of TensorFlare-1-Beta-2.
